CSS3 선택자

- 특정한 HTML 태그를 선택할 때 사용하는 기능
- 선택한 태그에 원하는 스타일이나 스크립트 적용 가능

- css 적용하는 3가지 방법
- 1. *.css : 외부에 파일 따로 만들어서 가져오는 방법
- 2. style : style태그로 적용하는 방법
- 3. h1 style="" 태그 안에 있는 syle속성에 바로 적용하는 방법
===> 보통은 *.css 방법을 권장!


[ 클래스 선택자 ]

- 특정 클래스가 있는 태그를 선택
- li class="font_color font_size" 처럼 중복해서 적용가능, 단 여러 속성을 쓸때는 띄어쓰기 필수
- 선택자 형태 : .클래스

[ id 선택자 ]

- 공간 분할 태그에 아이디 선택자를 사용하여 id 속성을 적용
- 특정 id 속성이 있는 태그를 선택. 특정 태그 하나를 선택할 때 사용
- 선택자 형태 : #아이디

- id 속성이 중복되면 해당 id 속성을 표시한 모든 태그에 스타 일이 적용됨.
- 스타일시트에서는 문제가 없지만
자바스크립트에서는 id 속성이 중복되면 문제가 발생하므로 주의!

[ 전체, 태그, 기본 선택자 ]

- 전체 선택자 : HTML 문서 안에 있는 태그를 모두 선택
- 전체 선택자 형태 : *
- 태그 선택자 : HTML 페이지 내부의 특정 태그를 모두 선택
- 태그 선택자 형태 : 태그
- 한꺼번에 여러 개의 선택자를 입력하여 스타일 속성 적용 -> 쉼표 사용
