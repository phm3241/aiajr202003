- 프로그래머가 임의로 서버쪽으로 전송해야하는 데이터나

- hidden 타입으로 하면 화면에 보이지 않게 되지만, name과 value값이 서버로 전송된다. - 아래의 경우, userName=scott & idx=2 로 전송된다.

버튼 표현 : submit, button, reset

submit은 form의 사용자 입력정보를 action속성의 전송방법으로 서버로 전송하는 기능.
value로 버튼 이름을 정할 수 있다.


버튼의 속성은은 페이지가 이동이 없고 그냥 버튼의 모양표현이다.
폼을 서버로 보내는 역할은 sumit이 한다.

sumit 버튼을 따로 쓰지 않고 button 태그로도 쓸 수있다. sumit 과 button의 기능은 동일 하지만 input type="button" 폼안의 버튼속성은 전송기능이 없음. 구별해서 사용해야한다.


input type = "reset" 초기화하게 되면 처음 설정해둔 상태로 돌아간다. 아래의 경우 입력값 cool이 초기화 상태가 된다.

select

- name, value 속성 기억하기! 이 이름의 이 값을 서버로 보내야하기 떄문에 중요!
- value에 값을 지정해주지 않으면, 사용자 선택값으로 전송된다.
- 사용자에게는 이름을 보여주게하고, 내부에서 처리할때는 식별할 수있는 코드값을 value속성을 사용할 수 있다.
